# ESP32 PIR Motion Sensor – Project Reflection

This project introduced me to motion detection using a Passive Infrared (PIR) sensor with the ESP32. Instead of controlling an output manually, I learned how the ESP32 can react automatically when it receives a digital signal from an external sensor.

One of the most important lessons I learned was how PIR sensors detect changes in infrared radiation produced by moving people or objects rather than measuring distance. I also learned how to configure the ESP32 to continuously monitor a digital input and respond immediately when motion is detected.

During this project, I improved my understanding of digital input devices, GPIO pin configuration, and real-time event detection. I also became more comfortable troubleshooting wiring issues, verifying sensor output with the Serial Monitor, and understanding why the PIR sensor requires a short warm-up period before providing reliable readings.

Completing this project strengthened my embedded systems skills by combining hardware and software into a practical security-style application. This project provides a strong foundation for future projects such as smart home automation, automatic lighting systems, security alarms, IoT monitoring devices, and camera-triggered motion detection systems.

Overall, this project increased my confidence working with sensors and demonstrated how embedded systems can interact with the physical world to make intelligent decisions.
