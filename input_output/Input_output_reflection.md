Reflection

This project helped me understand how an ESP32 can receive input from the real world instead of only controlling outputs. In my previous project, I programmed an LED to blink automatically. In this project, I learned how to use a push button to control the LED, which made the ESP32 react to user input.

While building the circuit, I learned how to wire a push button correctly on a breadboard, connect an LED with a current-limiting resistor, and use GPIO pins for both input and output. I also learned how digitalRead() checks whether the button is pressed and how an if statement lets the program decide whether the LED should turn on or off.

One challenge was making sure all the wires and components were connected correctly. Troubleshooting the wiring taught me to be patient and check each connection one step at a time. Seeing the LED respond to the button press showed me how hardware and software work together.

This project gave me a better understanding of digital inputs, GPIO, breadboard wiring, and basic embedded programming. It also gave me more confidence working with the ESP32, and I feel ready to move on to more advanced projects like LED pattern generators, traffic light controllers, and reaction-time games.
