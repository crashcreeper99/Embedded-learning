# ESP32 Digital Input and Output

## Overview
This project demonstrates how an ESP32 reads digital input from a push button and controls an LED based on the button's state. It introduces the interaction between sensors (inputs) and actuators (outputs).

## Features
- Reads a push button
- Controls an LED based on user input
- Demonstrates digital input and output
- Real-time hardware interaction

## Hardware
- ESP32 DevKit V1
- Breadboard
- Push button
- LED
- 220Ω resistor
- Jumper wires

## Software
- Visual Studio Code
- PlatformIO
- C++

## What I Learned
- Configuring GPIO inputs
- Using INPUT_PULLUP
- Reading digital signals
- Controlling outputs from user input
- Building interactive embedded systems

## Future Improvements
- Debounce the push button
- Toggle LED states
- Add multiple buttons
- Control several LEDs
- Implement a finite state machine

## Project Reflection
This project taught me how embedded systems respond to external events by reading digital inputs and controlling outputs. I learned the importance of reliable button reading, proper circuit wiring, and writing firmware that reacts to user interaction. Understanding digital input and output is a fundamental skill that will support future work in embedded systems, robotics, and IoT development.
