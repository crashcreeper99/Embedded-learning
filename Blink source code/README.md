# ESP32 Blink

## Overview
The ESP32 Blink project is my first embedded systems project using the ESP32 DevKit V1. The goal was to learn how to configure a GPIO pin as an output and control an LED through software.

## Features
- Turns an LED on and off repeatedly
- Demonstrates GPIO output control
- Introduces embedded programming fundamentals
- Written in C++ using PlatformIO

## Hardware
- ESP32 DevKit V1
- Breadboard
- LED
- 220Ω resistor
- Jumper wires

## Software
- Visual Studio Code
- PlatformIO
- C++

## What I Learned
- Setting up the ESP32 development environment
- Configuring GPIO pins
- Using digitalWrite()
- Using delay() for timing
- Uploading firmware to the ESP32

## Future Improvements
- Control multiple LEDs
- Replace delay() with millis()
- Add user input with a push button

## Project Reflection
This project introduced me to embedded systems programming and hardware control. I learned how software interacts directly with physical components through GPIO pins and gained experience wiring circuits, compiling firmware, and troubleshooting hardware connections. Completing this project established the foundation for more advanced ESP32 applications.
