# ESP32 Traffic Light Simulator

## Overview
This project simulates a real-world traffic light using an ESP32 DevKit V1. Three LEDs represent the red, yellow, and green traffic signals, cycling through a timed sequence to demonstrate basic embedded system control.

## Features
- Controls three LEDs independently
- Simulates a realistic traffic light sequence
- Uses GPIO outputs on the ESP32
- Demonstrates timing with delays
- Simple embedded firmware written in C++

## Hardware
- ESP32 DevKit V1
- Breadboard
- 3 LEDs
- 3 × 220Ω resistors
- Jumper wires
- USB cable

## Software
- Visual Studio Code
- PlatformIO
- C++

## Learning Outcomes
- GPIO programming
- Digital output control
- Circuit wiring
- Timing and sequencing
- Embedded systems fundamentals
