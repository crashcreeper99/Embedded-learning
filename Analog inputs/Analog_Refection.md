# Project Reflection – ESP32 Analog Inputs

Completing this project gave me a solid understanding of how analog inputs work on the ESP32. Before this project, I understood that sensors produced electrical signals, but I did not fully understand how a microcontroller interprets those signals. By connecting a potentiometer and observing the changing ADC values in the Serial Monitor, I saw how the ESP32 converts different voltage levels into numerical values between 0 and 4095.

One challenge was making sure the potentiometer was wired correctly and selecting an appropriate analog-capable GPIO pin. After correcting the wiring, the readings responded smoothly as I rotated the potentiometer.

This project strengthened my understanding of analog-to-digital conversion, sensor interfacing, and serial communication. These are core skills that will be essential for more advanced projects involving temperature sensors, light sensors, distance sensors, robotics, automation, and IoT devices.

Overall, this project was another important step toward becoming an embedded systems engineer. Each project builds on the previous one and increases my confidence in designing and programming real hardware with the ESP32.
