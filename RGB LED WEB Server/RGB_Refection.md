# Reflection – ESP32 RGB LED Web Server

This project was one of my most advanced ESP32 projects because it combined hardware control with networking and web development. Instead of controlling an LED using buttons or sensors, I created a web interface that allowed the RGB LED to be controlled wirelessly over Wi-Fi.

During the project I learned how the ESP32 acts as a web server, responds to HTTP requests, and uses PWM signals to generate different LED colors. I also gained experience troubleshooting software compatibility issues between different versions of the ESP32 Arduino Core, which improved my debugging skills.

Completing this project strengthened my understanding of Internet of Things (IoT) concepts and showed me how embedded systems can interact with users through a web browser. It also gave me confidence in integrating multiple technologies—including embedded programming, networking, HTML, and JavaScript—into a single application.

This project is an important step toward building more advanced IoT systems, smart home devices, and connected embedded applications for my engineering portfolio.
