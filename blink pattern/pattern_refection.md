# Project Reflection – LED Pattern Generator

Today I completed my LED Pattern Generator project using the ESP32. This project expanded on my previous Blink LED and Push Button LED projects by introducing multiple blinking patterns and organizing the code into reusable functions.

While building this project, I learned how to create different LED behaviors by adjusting timing with `delay()` and using `for` loops to repeat patterns. I also practiced writing functions to keep my program organized and easier to maintain. Creating patterns like slow blink, fast blink, double blink, SOS, and heartbeat helped me understand how embedded systems use timing and sequencing to control hardware.

One challenge was making sure each pattern flowed smoothly and that the delays produced the intended visual effect. Testing and adjusting the timing taught me the importance of debugging and incremental improvements in embedded programming.

Completing this project has strengthened my understanding of digital output control, loops, functions, and program structure. It also gave me more confidence working with the ESP32 and reinforced the value of writing clean, reusable code.

This is another step toward my goal of building a strong embedded systems portfolio and preparing for future engineering internships. My next goal is to build more advanced projects using multiple LEDs, sensors, and communication between hardware components.
