# ESP32 LED Pattern Generator

## Overview
This project expands on the basic blink project by creating multiple LED blinking patterns using an ESP32 DevKit V1. Different lighting sequences demonstrate how embedded systems coordinate outputs through software.

## Features
- Multiple LED blinking patterns
- Sequential lighting effects
- Alternating LED patterns
- Repeating programmed sequences

## Hardware
- ESP32 DevKit V1
- Breadboard
- LEDs
- 220Ω resistors
- Jumper wires

## Software
- Visual Studio Code
- PlatformIO
- C++

## What I Learned
- Controlling multiple GPIO pins
- Writing organized embedded code
- Timing multiple outputs
- Creating repeatable lighting sequences
- Debugging hardware connections

## Future Improvements
- Random LED patterns
- PWM brightness control
- Pattern selection with push buttons
- Non-blocking timing using millis()

## Project Reflection
Building the LED Pattern Generator helped me understand how embedded systems coordinate multiple outputs simultaneously. I gained experience writing structured firmware, organizing code for different lighting patterns, and troubleshooting both hardware and software issues. This project strengthened my understanding of timing, GPIO control, and program flow while preparing me for more advanced embedded applications.
